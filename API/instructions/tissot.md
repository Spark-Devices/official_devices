Clean flash ↓

- Boot TWRP via fastboot (Official TWRP 3.5.2 Preffered)
- Wipe Data/Cache/System
- Download ROM from the link above
- Flash ROM zip
- Flash TWRP (Optional)
- Format data
- Reboot and voila!

Updating to a newer build (dirty flash) ↓

- Update via OTA Updater, or
- Flash ROM zip (sideload)
- Reboot and voila!
